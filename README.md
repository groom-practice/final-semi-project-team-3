# Final Semi Project

본격적으로 팀 프로젝트 들어가기 전 마지막으로 협업 실습 프로젝트

## 프로젝트 실행 방법

<code>
$ npm install<br>
$ npm run dev
</code>

## 팀원 및 역할

|  이름  |       역할        |
| :----: | :---------------: |
| 이예도 |   로그인 페이지   |
| 신유승 | Posts 버튼 스크롤 |
| 권수영 | Posts 무한 스크롤 |
| 양아름 |    Posts Modal    |
| 김호영 |      Swiper       |
| 김기현 |       Photo       |
